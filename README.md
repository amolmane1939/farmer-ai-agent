# Farmer AI Agent

An AI-powered assistant to help farmers with crop-related issues including climate change, infrastructure, profitability, soil degradation, and pest/disease management.

## Features
- Mobile-friendly web interface (works on Android browsers)
- Quick access buttons for common farming issues
- Real-time chat with AI agent
- Covers: Climate, Soil, Pests, Profitability, Infrastructure

## Setup

1. Get Free Gemini API Key:
   - Go to: https://makersuite.google.com/app/apikey
   - Click "Create API Key"
   - Copy your key

2. Install Python dependencies:
```
pip install -r requirements.txt
```

3. Set your API key (Windows):
```
set GEMINI_API_KEY=your_api_key_here
```

Or create `.env` file:
```
GEMINI_API_KEY=your_api_key_here
```

4. Run the application:
```
python app.py
```

5. Access on your phone:
   - Find your computer's IP address
   - On your Android phone browser, go to: http://YOUR_IP:5000

## Usage

- Type your farming question or use quick buttons
- Ask about climate issues, soil problems, pest control, profitability, or infrastructure
- Get instant advice and recommendations

## Example Questions
- "How to deal with drought?"
- "My soil is losing fertility"
- "Pest problem in my crops"
- "How to increase profit?"
- "Need better storage facilities"
